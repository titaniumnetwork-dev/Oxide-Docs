Night Network is an organization of developers focused on creating innovative middleware and transport solutions for web proxies, with an emphasis on extensibility, user control, and powerful customization.

# Active Projects

## Services

- [**Space**](../services/space.mdx) - A modern proxy browser built for privacy and ease of use, known for its signature UI.
- [**DaydreamX**](../services/daydreamx.mdx) - An advanced proxy browser.

## Transports

- [**Reflux**](../transports/reflux.mdx) - A powerful, universal request/response middleware engine for BareMux-compatible web proxies with a dynamic plugin system. Developed under the Obsidian Developer Labs sub-organization.

# Relations to TN

Night Network is an organization partnered with TN because of its community and large userbase of 2M+ active users.
