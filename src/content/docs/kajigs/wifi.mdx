### Get your Wifi Password

<small><i>By Luphoria</i><br/></small>

#### Requirements
chrome://net-export and chrome://policy unblocked

#### Steps
1. Visit chrome://net-export
2. In "OPTIONS" set "Include raw bytes"
3. Click "Start Logging to Disk"
4. Visit chrome://policy
5. Click "Reload policies"
6. Go back to chrome://net-export and click "Stop logging"
7. Upload file to [this website](https://luphoria.com/netlog-policy-password-tool)

Video tutorial from Synaptic coming soon.
:::

### Connect to non-whitelisted networks

#### Requirements

- Chromebook
- Whitelisted Wi-Fi networks enabled

#### Steps

1. Turn off Wi-Fi
2. Turn Wi-Fi on then click "Join Wi-Fi network" as fast as possible
3. Input network details
4. Disable Wi-Fi
5. Enable Wi-Fi then click "Connect" as fast as possible

:::note
If you open "settings" and disable Wi-Fi you can change network settings
:::
